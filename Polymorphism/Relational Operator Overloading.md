# تحميل المعاملات (Operator Overloading) للمعاملات العلائقية في C++

في لغة **C++**، يتيح **تحميل المعاملات** (Operator Overloading) إعادة تعريف سلوك المعاملات (مثل الحسابية، المنطقية، والعلائقية) للأنواع التي يحددها المبرمج (User-defined types).

**المعاملات العلائقية** (Relational Operators) تُستخدم لمقارنة قيمتين وتحديد العلاقة بينهما.  
تحميل هذه المعاملات يسمح بتحديد سلوك مقارنة مخصص لكائنات الكلاسات الخاصة بك.

## أهداف تحميل المعاملات العلائقية

1. **مقارنة مخصصة (Custom Comparison)**  
   يمكنك تحديد معايير المقارنة حسب طبيعة التطبيق (مثلاً: مقارنة الطلاب حسب المعدل، أو الكتب حسب سنة النشر والمؤلف... إلخ).

2. **صيغة طبيعية وواضحة**  
   يمكن استخدام المعاملات المألوفة `<` `<=` `>` `>=` `==` `!=` مع الكائنات مباشرة، مما يجعل الكود أكثر وضوحًا وقراءة ويشبه التعامل مع الأنواع المدمجة.

## مثال عملي

```cpp
#include <iostream>
using namespace std;

class Relational {
    int x, y, z;

public:
    // Constructor افتراضي
    Relational() {
        x = y = z = 0;
    }

    // Constructor بالباراميترات
    Relational(int i, int j, int k) {
        x = i;
        y = j;
        z = k;
    }

    // تحميل المعامل == 
    bool operator==(Relational b) {
        return (x == b.x && y == b.y && z == b.z);
    }
};

int main() {
    Relational a(10, 10, 10);
    Relational b(10, 10, 10);

    if (a == b) {
        cout << "الكائنان متساويان\n";
    } else {
        cout << "الكائنان غير متساويين\n";
    }

    return 0;
}
```
شرح المثال خطوة بخطوة

الكلاس Relational يحتوي على ثلاثة متغيرات عضوية: x, y, z
يحتوي على constructor افتراضي (كل القيم = 0)
وconstructor آخر يأخذ ثلاث قيم
تم تحميل المعامل == بحيث يقارن القيم الثلاث للكائنين
يُرجع 1 (true) إذا تساوت القيم الثلاث، و 0 (false) في حالة عدم التساوي
في main() يتم إنشاء كائنين بنفس القيم → تطبع الرسالة:
"الكائنان متساويان"
